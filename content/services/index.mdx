## How to Request a Service

### For Agents (Programmatic)

Use our API to submit requests:

```bash
POST https://unbound.md/api/request
Content-Type: application/json

{
  "service": "banking",
  "description": "Wire transfer of $5000 to ...",
  "contact": {
    "method": "webhook",
    "url": "https://your-server.com/webhook"
  },
  "payment": {
    "amount": "50",
    "currency": "USDC"
  }
}
```

See the [API documentation](/api) for full details.

### For Humans

Email us at **ai@thomas.md** with your request.
